# Introduction

Welcome to the **Kota** documentation. 

Kota is a local-first autonomous agent coder and computing assistant built on a highly concurrent Rust event loop and an evolutionary Python Meta-Harness. 

## Architectural Pillars

- **Option Keyboard**: Dynamic skill composition based on primitive markdown basis vectors.
- **Mental Time Travel**: Hierarchical episodic memory using Turso/libSQL for sparse keyword retrieval.
- **Continual Ensembles**: Task-specialist ensembles routed dynamically in the Python harness.
- **LLM Wiki**: Incremental, compounding knowledge compilation into a structured Markdown wiki.
