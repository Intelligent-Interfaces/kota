# Interactive Notebooks

Kota features a suite of Jupyter notebooks for educational deep-dives and development:

- `notebooks/L1_Dynamic_Prompting.ipynb` - Explores modular skill trees and system prompt composition.
- `notebooks/L2_Memory_and_Turso.ipynb` - Explores Turso SQLite memory systems and message logs.
- `notebooks/L3_Skills_and_Observability.ipynb` - Deep dive into active tools and tracing.
- `notebooks/L4_Meta_Search.ipynb` - Explores evolution history and candidate metrics.
