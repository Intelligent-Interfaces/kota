Kota Documentation

Introduction

Welcome to the Kota documentation.

Kota is a local-first autonomous agent coder and computing assistant built on a highly concurrent Rust event loop and an evolutionary Python Meta-Harness.

Architectural Pillars

  • Option Keyboard: Dynamic skill composition based on primitive markdown basis vectors.
  • Mental Time Travel: Hierarchical episodic memory using Turso/libSQL for sparse keyword retrieval.
  • Continual Ensembles: Task-specialist ensembles routed dynamically in the Python harness.
  • LLM Wiki: Incremental, compounding knowledge compilation into a structured Markdown wiki.

Getting Started

This guide will walk you through setting up and running Kota locally.

Installation

First, make sure you have Rust and Cargo installed. Clone the repository and run:

cargo build --release

To install the documentation viewer locally, run:

cargo install mdbook

Running the Agent

Start your local LLM engine (e.g., Ollama, llama.cpp, or vLLM). Then run Kota:

cargo run -- --model qwen3:8b --api-url http://localhost:11434/v1 --workdir ~/my-project

Hotkeys

  • Enter: Send message
  • Ctrl+C: Quit
  • Ctrl+R: Reset conversation
  • PageUp/PageDown: Scroll output

Hierarchical Memory

Kota implements a two-tiered memory hierarchy to prevent context bloat:

  1. Working Memory: In-memory message list limited to the current session.
  2. Episodic Memory: SQL database (Turso/libSQL) that persists all conversational traces across sessions.

Fuzzy keyword matching retrieves previous relevant traces and injects them as episodic context, enabling "Mental Time Travel".

Option Keyboard & Skills

Agent behaviors in Kota are treated as linear combinations of markdown skill files stored in .kota_skills/.

The SkillComposer loads active skills and weighs them according to the selected AgentMode:

  • Coder: 1.0 Coder + 0.2 Eval
  • Research: 1.0 Research + 0.3 Coder
  • Cpe: 1.0 Cpe + 0.4 Architect
  • Architect: 1.0 Architect + 0.5 Cpe
  • Librarian: 1.0 Librarian + 0.4 Research

Interactive Notebooks

Kota features a suite of Jupyter notebooks for educational deep-dives and development:

  • notebooks/L1_Dynamic_Prompting.ipynb - Explores modular skill trees and system prompt composition.
  • notebooks/L2_Memory_and_Turso.ipynb - Explores Turso SQLite memory systems and message logs.
  • notebooks/L3_Skills_and_Observability.ipynb - Deep dive into active tools and tracing.
  • notebooks/L4_Meta_Search.ipynb - Explores evolution history and candidate metrics.
